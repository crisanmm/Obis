# Obis (RO Obesity Prevalence Visualizer) [M]

Sa se creeze o aplicatie Web de vizualizare a datelor referitoare la obezitatea populatiei – a se folosi datele din cadrul [BRFSS](https://www.cdc.gov/brfss/annual_data/annual_data.htm) – pe baza unui API REST/GraphQL propriu. Statisticile, comparatiile intre state, plus vizualizarile generate – minim 3 maniere – vor putea fi exportate in formatele CSV, WebP si SVG. Resurse suplimentare despre vizualizarea datelor: https://profs.info.uaic.ro/~busaco/teach/courses/cliw/web-film.html#week4.

## OLD

Sa se creeze o aplicatie Web de vizualizare a datelor referitoare la obezitatea populatiei – a se folosi [Romania Female obesity prevalence, 1960-2018](https://knoema.com/atlas/Romania/Female-obesity-prevalence) si [Romania Male obesity prevalence, 1960-2018](https://knoema.com/atlas/Romania/Male-obesity-prevalence) – pe baza unui API REST/GraphQL propriu. Statisticile, comparatiile cu alte natiuni, plus vizualizarile generate – minim 3 maniere – vor putea fi exportate in formatele CSV, WebP si SVG. Resurse suplimentare despre vizualizarea datelor: https://profs.info.uaic.ro/~busaco/teach/courses/cliw/web-film.html#week4.
